NilamFlow

Farm operations software built for Indian farms — from field to market in one platform.

NilamFlow is a multi-tenant SaaS application enabling farm owners and operators to run field-to-market operations from one integrated platform. It serves farms with multiple locations and workforces — seasonal pickers through to permanent staff — with harvest tracking, payroll, vendor dispatch, retail sales, inventory, and financial reporting.

  • Harvest session lifecycle — open → closed → payroll pending → paid, with per-worker weight and rate tracking
  • Component-based payroll — allowances, bonuses, deductions with draft → approved → paid workflow and PDF payslip export
  • Vendor dispatch and retail sales with credit limits, PDF invoices, and payment tracking
  • Razorpay subscription billing with tiered plans, dunning workflow, and 3-stage email reminders
  • AI crop price intelligence via Claude API with trend monitoring and source configuration

GraftAI

Let your users tell you what to build next.

GraftAI is an open-source Laravel package for multi-tenant SaaS platforms. Tenants describe what they need in plain English — GraftAI uses Claude AI to convert that into a live, scheduled data pipeline instantly. No code, no deployment. Over time it watches which pipelines tenants independently build, spots patterns, and surfaces them as one-click feature promotions — turning real usage into your product roadmap.

  • Plain English → live data pipeline via Claude AI — no code required
  • Scheduled execution with email and webhook notification triggers
  • Automatic pattern detection surfaces feature candidates across tenants
  • One-click promotion turns usage signals into versioned platform features

Behavioral Analytics

Portfolio Intelligence System

Behavioral telemetry for developer-AI collaboration — signal-driven and archetype-aware.

A behavioral telemetry platform that ingests raw development events — commits, edits, AI prompts, refactors — and processes them through a service-oriented pipeline to produce normalized signals, effort scores (0–10), and archetype classifications across 6 developer profiles.

  • 6 core signals aggregated over rolling 7-day and 30-day windows
  • Softmax-based archetype classification: Architect, AI Conductor, Refiner, and more
  • Vue 3 dashboard with real-time charts and live event streams
  • Simulation endpoint generating 18-session mock telemetry for local testing
  • Clean architecture — Form Request validation, dedicated service layer, cached signal storage

ComposerAI

Clinical documentation that writes itself — in seconds.

ComposerAI is a full-stack SaaS platform that automates clinical documentation for healthcare practitioners. Doctors record or upload consultation audio; the platform transcribes, structures, and generates complete clinical notes and letters — ready to review in seconds, tuned per medical specialty.

  • Real-time transcription — live audio streamed via WebSocket to OpenAI Whisper and LemonFox for instant transcripts
  • Multi-model LLM pipeline — GPT-4/5 and Gemini 2.5 with specialty-specific models per clinical section (HxPC, medications, examination, management plan)
  • Template & personality engine — customisable per specialty (psychiatry, orthopaedics, PT/OT) with RAG-based retrieval via Weaviate
  • Multi-tenant SaaS — organisation-scoped data isolation, role-based access, and per-party subscription enforcement
  • Stripe billing — usage-based limits, trial management, and full webhook lifecycle handling
