SaaS Platform

NilamFlow

A unified operations platform for Indian farmers and agri-businesses.

NilamFlow is a cloud-based SaaS platform purpose-built for Indian agriculture. It consolidates farm operations — harvest tracking, workforce management, inventory, vendor relations, and finances — into a single, role-based dashboard accessible from any device.

  • Harvest tracking — log sessions, record weights, and link produce to inventory in real time
  • Workforce & payroll — manage permanent and seasonal staff, attendance, leave, and salary disbursement
  • Inventory control — monitor stock levels, process goods, and record retail transactions
  • Live crop market prices sourced from data.gov.in government feeds
  • Secure subscription billing via Razorpay with role-based access and two-factor authentication
Laravel Cloud SaaS India-first Razorpay Multi-role

GraftAI

Feature development driven by real user behaviour, not internal assumptions.

GraftAI is an open-source Laravel package that enables multi-tenant SaaS platforms to discover, validate, and promote new features organically. Tenants describe requirements in plain English — GraftAI translates them into live, scheduled data pipelines via Claude AI. When tenant patterns converge, the system surfaces promotion candidates for a single-approval workflow that turns usage signals into versioned platform capabilities.

  • Natural language to live data pipeline — powered by Claude AI, no code required
  • Scheduled pipeline execution with configurable email and webhook notification triggers
  • Automated pattern analysis surfaces feature promotion candidates across tenant cohorts
  • Single-review approval converts usage signals into versioned, audited platform features
